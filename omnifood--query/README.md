## font-size

<br>

```css
@media (max-width: 75em) {
  html {
    font-size: 56.25%;
  }
}
```

```
기존 62.5% (10px)에서 56.25% (9px)로 바꾸면서 rem,em으로 설정한 모든 요소의 크기를 한번에 변경
```
